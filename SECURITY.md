# Security Policy

## Reporting Security Issues

If you discover a security issue within this repository, please open a private GitHub Security Advisory or contact the maintainers directly.

Please do not publish zero-day vulnerabilities through GitHub Issues.

## Supported Content

This repository focuses on defensive security.

It does not provide malware, exploit code, or offensive tooling.

## Responsible Disclosure

Please allow maintainers adequate time to investigate and address reported issues.
